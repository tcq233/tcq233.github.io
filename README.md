## 仓库

- 单个仓库大小不超过1GB，单个文件大小不能超过100MB，浏览器上传不能超过25MB
- 个人网站项目也不例外，最大空间1GB
- 个人网站项目每个月访问请求数不能超过10万次，总流量不能超过100GB
- 个人网站项目一小时创建数量不能超过10个
