## 简介

通过样式代码，可以绘制一些基础的图形。

- [CSS样式调试](https://c.runoob.com/front-end/61)
- [CSS样式教程](https://www.runoob.com/css3/css3-tutorial.html)

## 示例代码

样式代码：

````
<style type="text/css">
#circle {
	width: 100px;
	height: 100px;
	background: red;
}
</style>
````
